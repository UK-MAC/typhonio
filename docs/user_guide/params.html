Types and Parameters

C/C++ Types and Parameters

Look in the header files!

Fortran90 Types and Parameters

 

Types:

  integer(kind=TIO_FILEK)   Holds file ID
  integer(kind=TIO_OBJK)   Holds object ID
  integer(kind=TIO_ERRK)   Holds return code - see below for values
  integer(kind=TIO_SIZEK)   Holds size values
  integer(kind=TIO_STEPK)   Holds State step number values
  real(kind=TIO_TIMEK)   Holds time values
  integer(kind=TIO_SHAPEK)   Holds unstructured shapes
 

Limits:

  TIO_STRLEN_F   Maximum length of name, unit, label and data strings
  TIO_MAXDIMS_F   maximum number of dimensions allowed in a variable
  TIO_NULL_F   General NULL value
 

Error return codes:

  TIO_SUCCESS_F   0
  TIO_FAIL_F   -1
  TIO_ERR_INVALID_FILEID_F   -2
  TIO_ERR_INVALID_OBJECTID_F   -3
  TIO_ERR_INVALID_DATATYPE_F   -4
  TIO_ERR_INVALID_DIMS_F   -5
  TIO_ERR_INVALID_ARG_F   -6
  TIO_ERR_INVALID_RANGE_F   -7
  TIO_ERR_INVALID_INDEX_F   -8
  TIO_ERR_INVALID_NAME_F   -9
  TIO_ERR_INVALID_STRING_F   -10
  TIO_ERR_INVALID_OPERATION_F   -11
  TIO_ERR_INVALID_CONVERSION_F   -12
  TIO_ERR_INVALID_SECTIONID_F   -13
  TIO_ERR_FILE_CREATE_F   -20
  TIO_ERR_FILE_OPEN_F   -21
  TIO_ERR_FILE_CLOSE_F   -22
  TIO_ERR_FILE_FLUSH_F   -23
  TIO_ERR_FILE_NOT_EXIST_F   -24
  TIO_ERR_FILE_NOT_WRITEABLE_F   -30
  TIO_ERR_FILE_NOT_READABLE_F   -31
  TIO_ERR_FILE_NOT_CLOSEABLE_F   -32
  TIO_ERR_OBJECT_NOT_WRITEABLE_F   -33
  TIO_ERR_OBJECT_NOT_READABLE_F   -34
  TIO_ERR_OBJECT_NOT_CLOSEABLE_F   -35
  TIO_ERR_OBJECT_CREATE_F   -40
  TIO_ERR_OBJECT_OPEN_F   -41
  TIO_ERR_OBJECT_CLOSE_F   -42
  TIO_ERR_OBJECT_WRITE_F   -43
  TIO_ERR_OBJECT_READ_F   -44
  TIO_ERR_OBJECT_ALREADY_EXIST_F   -45
  TIO_ERR_OBJECT_NOT_EXIST_F   -46
  TIO_ERR_VERSION_MISMATCH_F   -80
  TIO_ERR_LIBRARY_MISMATCH_F   -81
  TIO_ERR_UNSUPPORTED_FILETYPE_F   -82
  TIO_ERR_F90_API_F   -90
  TIO_ERR_INT_F   -999
 

Datatypes:

  TIO_DATATYPE_NULL_F   Null value
  TIO_REAL4_F   Datatype for 4-byte real, ie. real(kind=4)
  TIO_REAL8_F   Datatype for 8-byte real, ie. real(kind=8) or double precision
  TIO_INTEGER4_F   Datatype for 4-byte integer, ie. integer(kind=4)
  TIO_INTEGER8_F   Datatype for 8-byte integer, ie. integer(kind=8)
  TIO_LOGICAL_F   Datatype for 4-byte logical, ie. logical
  TIO_CHAR_F   Datatype for character, ie. character
  TIO_STRING_F   Datatype for character string, ie. character(len=TIO_STRLEN_F)
  TIO_SIZE_F   Datatype for TyphonIO Sizes, ie. integer(kind=TIO_SIZEK)
 

Type classes:

  TIO_TYPECLASS_NULL_F   Null value
  TIO_TYPECLASS_INTEGER_F   TypeClass for integer datatypes
  TIO_TYPECLASS_REAL_F   TypeClass for real datatypes
  TIO_TYPECLASS_STRING_F   TypeClass for string datatypes
 

Dimensions:

  TIO_0D_F   (=0)
  TIO_1D_F   (=1)
  TIO_2D_F   (=2)
  TIO_3D_F   (=3)
 

Ghosts:

  TIO_GHOSTS_NULL_F   Null value
  TIO_GHOSTS_NONE_F   Signifies none of the mesh ghosts
  TIO_GHOSTS_ALL_F   Signifies all of the mesh ghosts
 

File Access parameters:

  TIO_ACC_NULL_F   Null value
  TIO_ACC_CREATE_F   Create new file and fail if already exists
  TIO_ACC_REPLACE_F   Create new file replacing any existing file
  TIO_ACC_READWRITE_F   Open file for reading and modifying
  TIO_ACC_READONLY_F   Open file for reading only
 

Meshtype parameters:

  TIO_MESH_NULL_F   Null value
  TIO_MESH_QUAD_F = TIO_MESH_QUAD_COLINEAR_F  
  TIO_MESH_QUAD_COLINEAR_F  
  TIO_MESH_QUAD_NONCOLINEAR_F  
  TIO_MESH_UNSTRUCT_F  
  TIO_MESH_POINT_F  
 

Coordtype parameters:

  TIO_COORD_NULL_F   Null value
  TIO_COORD_CARTESIAN_F   Cartesian co-ordinates {i,j,k} = {x,y,z}
  TIO_COORD_SPHERICAL_F   Spherical polar co-ordinates {i,j,k} = {r,theta,phi}
  TIO_COORD_CYLINDRICAL_F   Cylindrical polar co-ordinates {i,j,k} = {r,phi,z}
  TIO_COORD_CYL_AXISYMMETRIC_F   Cylinder axi-symmetric co-ordinates {i,j} = {z,r}
 

Shape parameters:

  TIO_SHAPE_NULL_F   Null value
  TIO_SHAPE_POINT1_F   0D Point shape with 1 node
  TIO_SHAPE_BAR2_F   1D Bar shape with 2 nodes
  TIO_SHAPE_TRI3_F   2D Triangle shape with 3 nodes & 3 edges
  TIO_SHAPE_QUAD4_F   2D Quadrilateral shape with 4 nodes & 4 edges
  TIO_SHAPE_TET4_F   3D Tetrahedral shape with 4 nodes & 6 edges
  TIO_SHAPE_PYR5_F   3D Pyramid shape with 5 nodes & 8 edges
  TIO_SHAPE_WEDGE6_F   3D Wedge shape with 6 nodes & 9 edges
  TIO_SHAPE_HEX8_F   3D Hexahedral shape with 6 nodes & 12 edges
 

Centring parameters:

  TIO_CENTRE_NULL_F   Null value
  TIO_CENTRE_CELL_F   Cell-centred
  TIO_CENTRE_NODE_F   Node-centred
  TIO_CENTRE_EDGE_I_F   Edge-centred on I-axis
  TIO_CENTRE_EDGE_J_F   Edge-centred on J-axis
  TIO_CENTRE_EDGE_K_F   Edge-centred on K-axis
  TIO_CENTRE_FACE_I_F   Face-centred on I-axis
  TIO_CENTRE_FACE_J_F   Face-centred on J-axis
  TIO_CENTRE_FACE_K_F   Face-centred on K-axis
 

Data transfer parameters:

  TIO_XFER_NULL_F   Null value
  TIO_XFER_INDEPENDENT_F   Independent data transfer
  TIO_XFER_COLLECTIVE_F   Collective data transfer - all processes must call simultaneously
 

Verbose mode parameters:

  TIO_VERBOSE_NULL_F   Null value
  TIO_VERBOSE_OFF_F   Turn off all TyphonIO verbose output
  TIO_VERBOSE_ALL_F   Turn on verbose output from all processes
  TIO_VERBOSE_MASTER_F   Turn on verbose output only on the Master process (with MPI rank == 0)
