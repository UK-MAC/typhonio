TyphonIO: Quant Routines

TyphonIO Quants are objects that can hold any other mesh-wide data that isn't the material. The mixed look-up arrays within the material also act as look-ups for the quantity mixed, so the qmix array should be adhere to the ncomp and ptr array.

TIO_List_Quants
PURPOSE
Returns the number of Quants within a TyphonIO file or object and their names
SYNOPSIS
C/C++:
 TIO_t TIO_List_Quants( const TIO_File_t   fileID,
                        const TIO_Object_t meshID,
                        TIO_Size_t         *nobj,
                        char               names[][TIO_STRLEN] );
Fortran90
 integer(kind=TIO_ERRK) function TIO_List_Quants_f( fileID, meshID, nobj, names )

   integer(kind=TIO_FILEK),        intent(in)            :: fileID
   integer(kind=TIO_OBJK),         intent(in)            :: meshID
   integer(kind=TIO_SIZEK),        intent(out), optional :: nobj
   character(len=*), dimension(:), intent(out), optional :: names
PARAMETERS
IN fileID TyphonIO File ID
IN meshID TyphonIO Mesh ID
OUT nobj Number of Quants
OUT names Array of names
RETURNS
TyphonIO return code
DESCRIPTION

Does not have to be called collectively.

C/C++ Notes:
names can be NULL


TIO_Create_Quant
PURPOSE
Creates a new Quant within a Mesh
SYNOPSIS
C/C++:
 TIO_t TIO_Create_Quant( const TIO_File_t   fileID,
                         const TIO_Object_t meshID,
                         const char         name[],
                         TIO_Object_t       *quantID,
                         const TIO_Data_t   datatype,
                         const TIO_Centre_t centring,
                         const TIO_Size_t   nghosts,
                         const TIO_Bool_t   ismixed,
                         const char         units[] );
Fortran90
 integer(kind=TIO_ERRK) function TIO_Create_Quant_f( fileID, meshID, name, quantID,
                                                     datatype, centring, nghosts,
                                                     ismixed, units )

   integer(kind=TIO_FILEK), intent(in)  :: fileID
   integer(kind=TIO_OBJK),  intent(in)  :: meshID
   character(len=*),        intent(in)  :: name
   integer(kind=TIO_OBJK),  intent(out) :: quantID
   integer(kind=TIO_IPK),   intent(in)  :: datatype, centring
   integer(kind=TIO_SIZEK), intent(in)  :: nghosts
   logical(kind=TIO_LPK),   intent(in)  :: ismixed
   character(len=*),        intent(in)  :: units
PARAMETERS
IN fileID TyphonIO File ID
IN meshID TyphonIO Mesh ID
IN name Name for Quant
OUT quantID ID of new Quant
IN datatype TyphonIO datatype of Quant
IN centring Data centring of Quant
IN nghosts Number of ghosts in Quant
IN ismixed Flag whether Quant is mixed
IN units String of Quant units
RETURNS
TyphonIO return code
DESCRIPTION

ismixed is global

Must be called collectively.
